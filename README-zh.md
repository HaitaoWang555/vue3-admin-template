# vue-admin-template

> 这是一个极简的 vue admin 管理后台。它只包含了 Element UI & axios & iconfont & permission control & lint，这些搭建后台必要的东西。

## Build Setup

```bash
# 克隆项目
git clone https://github.com/HaitaoWang555/vue3-admin-template.git

# 进入项目目录
cd vue3-admin-template

# 安装pnpm
npm i -g pnpm

# 安装依赖
pnpm install

# 启动服务
pnpm dev
```

浏览器访问 [http://localhost:9528](http://localhost:9528)

## 发布

```bash
# 构建测试环境
pnpm build:stage

# 构建生产环境
pnpm build:prod
```

## 其它

```bash
# 预览发布环境效果
pnpm preview

# 预览发布环境效果 + 静态资源分析
pnpm preview -- --report

# 代码格式检查
pnpm lint

# 代码格式检查并自动修复
pnpm lint -- --fix
```
